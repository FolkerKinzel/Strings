      Remarks

        Note

      In the .NET-Framework 4.5 and .NET Standard 2.0 versions of the nuget package, the method has to allocate a new Array. For this reason it is recommended, to use the overloads Encode(Byte, Int32, Int32, Base64FormattingOptions) or Encode(Byte, Base64FormattingOptions) when supporting old framework versions.
